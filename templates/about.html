Acerca de

Este videojuego fue el producto final de un proyecto de universidad. Este programa fue creado desde cero en Java y con mucha dedicación.

¿Cómo se juega?

El usuario tendrá control de un carro que se puede mover hacia el lado izquierdo y derecho, así como también arriba y abajo. De esta forma el jugador puede esquivar a los rivales con el fin de no perder vidas. Si el jugador recoge dos animales de la carretera, obtendrá una vida adicional. La dificultad del juego aumentará gradualmente cada 20 segundos, poniendo a prueba las habilidades del jugador. La partida terminará si el jugador pierde todas las vidas o después de 100 segundos de juego.

Roles de equipo

El desarrollo del juego se dividió en 3 roles:

  • El backend, encargado de la lógica del código, para el correcto funcionamiento del videojuego, el cual no es visible para el usuario. Se centra en que las acciones que realice el usuario por el teclaso y mouse, sean recibidas por el programa, para ser leídas y arrojar otra acción como respuesta.
  • El frontend, encargado del diseño y distribución de los elementos visibles para el usuario, como por ejemplo la interfaz y los botones interactivos que la componen, así mismo el diseño de mapas y animales, carros y sonidos. Todo esto para mejorar la experiencia del usuario.
  • El integrador, encargado de liderar, supervisar, definir los objetivos a realizar y sincronizar el backend del frontend. El integrador es principal apoyo del equipo por ser quien ayuda en ambas áreas (código y diseño), por ello, si hay una dificultad de desarrollo, es quien ofrece una alternativa de implementación.
