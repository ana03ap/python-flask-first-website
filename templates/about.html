Acerca de iPensum

Esta aplicación tiene como objetivo simplificar la manera en la cual se información acerca de la carrera de Ing. de sistemas en Uninorte

¿Cómo funciona?

El usuario tendrá la oportunidad de entrar a la aplicación como un invitado o iniciar sesión como estudiante ingresando, fundamentalmente, su código estudiantil. Seguido a esto, iPensum le permtirá seleccionar qué desea buscar; las opciones de busqueda son: malla curicular, información de materias por semestre, las materias favoritas de algunos estudiantes de la carrera y también las actividades extracurriculares, con las que cuenta la Universidad

¿Cómo y qué tener en cuenta para descargar la aplicación?

Lo más importante para tener en cuenta es :

  • Tener una versión de Python mayor o igual que 3.10.4
  • Contar con un buen procesador y buena definición de imagen para que la aplicación se logre visualizar mejor
  • Tener acceso a una red WiFi

Para descargar la aplicación solo debe seleccionar el botón en la parte inferior y se le redirecionará a la pagina :
